This method returns a list of platforms records. For official documentation of the
[endpoint](https://www.mobygames.com/info/api/#platforms), please refer to the MobyGames
documentation.

## Parameters

This method does not require any parameters.

## Example

```js {4}
import { MobyGames } from '@untidy/mobygames';

const client = new MobyGames('access api key');
const data = await client.platforms();
```
