<Tag apiPath={'/v1/games/:game_id/platforms/:platform_id'} />

This method returns a single game record on the specified platform. For official documentation of the
[endpoint](https://www.mobygames.com/info/api/#gamesgame_idplatformsplatform_id), please refer to the MobyGames
documentation.

## Parameters

| Params     | Type     | Required | Description       |
| ---------- | -------- | :------: | ----------------- |
| gameId     | `string` |   Yes    | The game `id`     |
| platformId | `string` |   Yes    | The platform `id` |

## Example

```js {4}
import { MobyGames } from '@untidy/mobygames';

const client = new MobyGames('access api key');
const data = await client.gamePlatformById('120555', '141');
```
