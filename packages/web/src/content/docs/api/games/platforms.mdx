This method returns a list of platforms records for which a game was released. For official documentation of the
[endpoint](https://www.mobygames.com/info/api/#gamesgame_idplatforms), please refer to the MobyGames
documentation.

## Parameters

| Params | Type     | Required | Description   |
| ------ | -------- | :------: | ------------- |
| id     | `string` |   Yes    | The game `id` |

## Example

```js {4}
import { MobyGames } from '@untidy/mobygames';

const client = new MobyGames('access api key');
const data = await client.gamePlatforms('16373');
```
