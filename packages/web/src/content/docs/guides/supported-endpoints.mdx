In version 1 of the official documentation for [MobyGames API](https://www.mobygames.com/info/api/), the status of each endpoint supported by `@untidy/mobygames` is indicated by the following emojis:

:::note

- &#128683;: This will never be implemented
- &#128284;: In progress
- &#10004;: Supported

:::
