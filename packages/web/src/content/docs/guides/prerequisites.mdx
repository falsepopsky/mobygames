---
title: Prerequisites
---

To successfully utilize this package, users must meet the following prerequisites:

- **Node.js:** The user must have [Node.js](https://nodejs.org) installed on their system with a
  minimum version of 18.19.1 or 20.8.1.

- **ESM:** When consuming this package, add the following line to your `package.json` file:

  ```
  "type": "module"
  ```

- **API key:** Obtain a free API key access from your [MobyGames](https://www.mobygames.com/) account. This will be required for accessing their API.

  :::tip

  If you'd like access to the free API (not-for-profit use), create a MobyGames account and then visit your profile page. Click the 'API' link under your user name to sign up for an API key.

  :::
